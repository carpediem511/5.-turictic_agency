## Проект "Туристическая Компания"

### ОСНОВНАЯ ИДЕЯ:

Сайт предоставляет информацию о доступных турах, их фильтрацию по различным критериям, возможностью добавления в избранное и удобной системой бронирования.

### ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА:

**Список Туров: отображение списка туров с информацией о стране, отеле, продолжительности, дате начала и окончания поездки и стоимости.
**Фильтрация: возможность фильтрации туров по стране, рейтингу, продолжительности.
**Сортировка: возможность сортировки туров по стоимости (возрастающей и убывающей).
**Добавить в Избранное: каждый тур снабжен кнопкой "Добавить в Избранное", позволяющей пользователям сохранять интересные туры для дальнейшего просмотра.
**Забронировать: каждый тур снабжен кнопкой "Забронировать", позволяющей пользователю оформить бронь.
**Модальное Окно: появление модального окна при нажатии на кнопку "Забронировать" с отображением деталей тура и формы для заполнения данных бронирования.

### ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ:

Frontend:

- HTML, CSS, JavaScript
- Фреймворк: Tailwind CSS
- Библиотеки: date-fns, sweetalert2 
- Сборка проекта: Parcel
- Веб-запросы: Fetch API для отправки GET и POST запросов на сервер

Backend (API):

- Работа с сервером по URL: https://www.bit-by-bit.ru/api/student-projects/tours
- Получение списка туров (GET запросы)
- Фильтрация туров по стране, рейтингу и цене
- Бронирование тура (POST запрос)

### ОСОБЕННОСТИ ПРОЕКТА:

- динамичная загрузка данных: использование fetch-запросов для получения списка туров с сервера.
- фильтрация и сортировка: реализация функционала фильтрации туров по стране, рейтингу, продолжительности тура и цене.
- модальные окна: использование модальных окон для улучшенного взаимодействия пользователя с сайтом.
- обработка бронирования: возможность отправки данных формы бронирования на сервер через post-запрос.

### ИНСТРУКЦИИ ПО УСТАНОВКЕ:

1. Клонировать репозиторий с помощью git clone.
2. Установить зависимости с помощью yarn install.
3. Запустить проект с помощью yarn start.
Перейти по адресу http://localhost:1234 в браузере.
